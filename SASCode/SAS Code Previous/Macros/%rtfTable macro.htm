All,

The RTF ODS engine in SAS version 9.2 introduced several new features that have been causing problems with my %rtfTable macro. Most notably, attempts to set the margins are ignored and trying to change the portrait/landscape orientation in a single document causes SAS to crash.

 

I’ve attached a 9.2-specific upgrade that fixes the margins problem. I have not been able to figure out what the issue is with the landscape/portrait issue. For now, do not try to change the orientation in the middle of an %rtfTable macro call.

 

I am no longer using this macro on a regular basis. For my reference, are you all still using it? Have you distributed it to others (I know that a couple of people in China oral care are)? I could put some effort into trying to update it to take advantage of improvements to the ods engine in 9.2, but I’m not sure if it would be worth the effort.
